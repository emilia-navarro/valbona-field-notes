# Topographic Metrics: Theth–Valbonë / Valbonë–Theth Corridor

## Purpose

This file records topographic and navigation notes for the classic Albanian Alps crossing between Theth and Valbonë via Qafa e Valbonës / Valbona Pass.

The route can be walked in either direction. Metrics differ slightly depending on the exact guesthouse, roadhead, trailhead, and GPS source used. Treat all numbers as route-planning values rather than survey-grade measurements.

## Reference Direction: Theth → Valbonë

The referenced trekking-stage guide gives the following values for the Theth → Valbonë direction:

- **Distance:** 16.4 km.
- **Elevation gain:** +1,175 m.
- **Elevation loss:** -940 m.
- **Estimated time:** 7 hours.
- **Difficulty:** moderate.
- **High point in profile:** 1,854 m.
- **Valbona Pass / Qafa e Valbonës:** commonly cited around 1,795 m.

## Reverse Direction: Valbonë → Theth

For the Valbonë → Theth direction, use the same corridor but invert the ascent/descent logic according to the actual start point. Guesthouses in Valbonë, Rragam, and the Fusha e Gjesë roadhead can shift the effective route length and elevation.

Do not publish a single exact “Rragam to Theth” value unless it is tied to a named GPX track.

## Navigation Notes

- The route follows an established historic mountain path / mule-track corridor.
- It is generally considered a hike rather than a technical climb.
- Exposure, wind, heat, dehydration, thunderstorms, and residual snow can materially change the risk profile.
- Waymarking and seasonal huts can change; carry offline maps and confirm with local guesthouses.

## Seasonal Window

A conservative planning window is late spring / early summer through early autumn. Snow can persist near the pass into June, and early or late season conditions require local confirmation. Do not state “safe from June 15 to October 1” as a universal rule; use it only as a planning heuristic and verify in the field.

## Publication Warning

Avoid over-precision. Values such as “32% peak incline over the final 1.2 km” should not be published unless derived from a stated DEM/GPX method and reproducible analysis.
